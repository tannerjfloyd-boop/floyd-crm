# Floyd CRM — Unified Mortgage Command Center

Personal CRM for mortgage loan officers. Built for Tanner Floyd.

## Stack
- **Frontend:** Lovable (React + Tailwind + shadcn/ui)
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **Automation:** Make.com webhooks (Phase 3)

## Features
- Unified contacts (leads, past clients, referral partners, builders)
- Lead inbox with Kanban board
- Active pipeline tracker with lock status monitoring
- Referral partner management (A/B/C tiers)
- Past client reactivation engine
- Activity timeline for every contact
- Rate feed integration (planned)

## Setup
1. Run `supabase/schema.sql` in your Supabase SQL Editor
2. Connect Lovable to your Supabase instance
3. Set environment variables (see `.env.example`)
